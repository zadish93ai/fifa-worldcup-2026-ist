# 🏆 Live-Syncing FIFA World Cup 2026 IST Calendar

An automated, live-updating calendar file (`.ics`) for all 104 matches of the FIFA World Cup 2026, automatically localized to **Indian Standard Time (IST)**.

## ✨ Features
- **🕒 Fully Localized:** All match kickoffs are converted automatically from North American time zones to Indian Standard Time (IST).
- **⚽ Live Score Updates:** The calendar dynamically pulls live scores every 3 hours. Concluded matches automatically update their titles (e.g., `🇲🇽 Mexico 2 - 1 South Africa 🇿🇦 (FT)`) right inside your calendar app!
- **🔄 Zero Maintenance:** Powered entirely by a cloud pipeline via GitHub Actions and the openfootball data feed. Subscribe once and stay informed on your phone or laptop.

## 📅 How to Subscribe (Google, Apple, Outlook)

Copy the official public calendar URL below:
```text
https://github.io
```

### 📱 Import into Google Calendar:
1. Open [Google Calendar](https://google.com) in a browser.
2. On the left side, click the **`+`** icon next to **"Other calendars"**.
3. Select **"From URL"**.
4. Paste the link above and click **"Add calendar"**.

### 🍏 Import into Apple Calendar (Mac/iPhone):
1. Open the Calendar app.
2. Go to **File > New Calendar Subscription...**
3. Paste the URL link and click **Subscribe**.
